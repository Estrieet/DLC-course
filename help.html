Help and Support

Clear instructions for using this website on phones, tablets, and computers.

How to use the website

Tap one of the top buttons to open a page. Use the big green buttons for the main actions.

Navigating on a phone

Scroll down slowly. Tap the large buttons and links. The page fits small screens automatically.

Getting help

If a page does not load, refresh the browser. The website works offline once it has loaded.
